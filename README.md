# Cloud App Deployment

A simple Python Flask web app deployed on AWS EC2 using Ubuntu, Gunicorn, and Nginx.

## Live Demo
http://3.138.194.188

## Tech Used
- Python
- Flask
- AWS EC2
- Ubuntu Linux
- Gunicorn
- Nginx
- systemd

## What I Did
- Built a Flask app locally
- Deployed it to AWS EC2
- Configured security groups (ports 22, 80, 5000)
- Set up Gunicorn + Nginx
- Created a systemd service
- Debugged real deployment issues (firewall, permissions, socket access)
